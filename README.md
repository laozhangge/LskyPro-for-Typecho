# 兰空图床上传插件 (LskyPro for Typecho)

这是一个Typecho插件，允许用户在编辑文章时直接上传图片到兰空图床(LskyPro)，无需手动上传后再复制链接。

通过WordPress兰空图床插件复刻而来，支持粘贴上传和存储策略/相册选择。

## 主要功能

- 在Typecho编辑器中添加兰空图床上传面板
- 支持拖拽和点击上传图片
- 支持多图片同时上传
- 支持配置上传权限（公开/私有）
- 限制上传文件大小和类型
- 支持编辑器粘贴上传 - 直接从剪贴板粘贴图片自动上传到兰空图床
- 存储策略选择 - 测试连接后自动加载存储策略列表
- 相册选择 - 可选择上传到指定相册
- 支持V1和V2版本兰空图床

## 安装方法

1. 下载插件压缩包
2. 解压到 `/usr/plugins/` 目录，文件夹名为 `LskyPro`
3. 在Typecho后台"控制台" > "插件"中启用插件
4. 点击"设置"配置插件

## 配置说明

1. 插件启用后，在"控制台" > "插件" > "兰空图床上传" > "设置"
2. 配置以下信息：
   - API网址：兰空图床的域名，如 `https://www.lsky.pro`
   - API Tokens：在兰空图床后台获取的API令牌
   - API版本：选择V1或V2版本
   - 图片权限：选择上传图片为公开还是私有
   - 存储策略：测试连接后自动加载
   - 上传相册：测试连接后自动加载
   - 最大上传大小：限制上传图片的大小
   - 允许的图片类型：限制可上传的图片格式

## 使用方法

### 编辑器上传面板

1. 在编辑文章时，页面下方会出现"兰空图床上传"面板
2. 点击上传区域选择图片
3. 图片上传成功后会显示URL、HTML和Markdown格式的链接
4. 复制链接粘贴到文章中

### 粘贴上传

1. 复制图片到剪贴板（如截图）
2. 在编辑器中按 `Ctrl+V`（Mac为 `Cmd+V`）粘贴
3. 图片会自动上传到兰空图床并插入到编辑器中

## 常见问题

**Q: 上传按钮没有反应？**
A: 请确认已在插件设置中正确配置API网址和Tokens，并检查网络连接。

**Q: 上传图片失败？**
A: 可能原因：图片太大、网络问题或API设置错误，请检查兰空图床后台是否能正常上传。

**Q: 粘贴上传不工作？**
A: 请确认使用的是标准的Typecho编辑器，并检查浏览器控制台是否有错误信息。
